## Development
### Setup
```sh
$ npm install
$ npm run flow-typed
```

### Development
```sh
$ npm run start:renderer
$ npm run start:main
```

### Production
```sh
$ npm run build
```

### Create The Package
### OSX
```sh
$ npm run package:osx
```
